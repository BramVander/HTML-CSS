HTML1

HTML is a markup language. It is used to give rise to structure in text by marking the text. Declaring a line as a heading element will display it as a heading. The same goes for a paragraph, an image, a link, etcetera, etcetera.

CSS2

CSS is a language for the lay-out. It is used to give styling to certain text or HTML elements. It enhances readability when headings are bold, if there is use of color or when certain things are lined out. With both of these tools you are able to build a full webpage!3
