HTML

HTML is een taal om tekst te markeren. Het wordt gebruikt om tekst te voorzien van structuur: dit is het eerste hoofdstuk, deze regel is een titel, deze alinea is een paragraaf, dit woord is een link, enzovoorts enzovoorts.

CSS

CSS is dan weer een lay-out taal en wordt gebruikt om opmaak aan tekst te geven. Het draagt bij aan leesbaarheid als titels dikgedrukt zijn of dat er kleur in de tekst zichtbaar is.
